# 🚴 Shanghai Ride / 上海骑行

**上海版 Strava + Komoot + Zwift** - 骑行社交 + 路线规划 + 数据追踪 + 虚拟骑行

---

## 🎯 产品愿景

> "上海版欧美骑行社交 App：记录 + 挑战 + 找搭子 + 虚拟沉浸"

### 目标用户
- 上海 20-45 岁白领
- 休闲/公路骑行爱好者
- 追求社交和数据的骑行用户

### 核心价值
| 维度 | 价值 |
|------|------|
| **记录** | GPS 轨迹、速度、海拔、卡路里 |
| **挑战** | Segments 路段排行榜 (KOM/QOM) |
| **社交** | 找搭子、车队、Feed 分享 |
| **沉浸** | 虚拟骑行（外滩/崇明岛实景） |

## 🚀 MVP 核心功能（第一优先级）

| 功能模块 | 说明 | 优先级 |
|----------|------|--------|
| 用户系统 | 微信/手机号/Apple ID 登录 | P0 |
| GPS 记录 | 距离、速度、海拔、轨迹、卡路里 | P0 |
| 路线规划 | 高德/百度地图集成 + 离线支持 | P0 |
| 社交 Feed | 发布照片/视频/成就，点赞评论 | P0 |
| Segments | 热门路段排行榜（苏州河/外滩/佘山） | P0 |
| 数据分析 | 周/月总结、热力图 | P0 |
| 找搭子 | 创建/加入车队，匹配用户 | P0 |

## 🛠️ 技术栈推荐

| 层级 | 技术选择 | 理由 |
|------|----------|------|
| **前端** | Flutter | iOS+Android 一套代码，性能好 |
| **后端** | Node.js + Go | 快速开发 + 高性能 |
| **数据库** | PostgreSQL + Redis | 轨迹数据 + 实时排行 |
| **地图** | 高德 API + OSM | 本地化 + 开源补充 |
| **云服务** | 阿里云 | 国内访问快、合规 |
| **实时** | WebSocket | 团骑实时位置 |
