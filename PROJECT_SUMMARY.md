## 📱 5 个核心页面

### 1️⃣ 首页 (Home)
- 天气卡片（温度、湿度、风速、AQI）
- 本周统计（骑行次数、距离、时间、消耗）
- 热门路段（苏州河、外滩、佘山）
- 动态 Feed（点赞、评论、分享）

### 2️⃣ 骑行记录 (Ride)
- 模拟地图区域
- 实时数据（速度、距离、时间）
- 开始/暂停/结束控制
- 骑行完成总结

### 3️⃣ 路线规划 (Route)
- 搜索栏
- 推荐路线列表（苏州河、外滩、佘山）
- 热门目的地网格
- 底部快捷操作

### 4️⃣ 挑战 (Challenges)
- Tab 切换（路段/徽章/活动）
- 路段挑战卡片（KOM/QOM 记录）
- 成就徽章系统
- 活动报名

### 5️⃣ 个人中心 (Profile)
- 用户信息卡片
- 数据统计（总距离、时间、消耗）
- 功能菜单（骑行/社交/其他）

## 🚀 如何运行

### 1. 安装 Flutter
如果还没安装：
```bash
# macOS
brew install --cask flutter
```

### 2. 进入项目目录
```bash
cd /Users/mao/clawd/shanghai-ride
```

### 3. 安装依赖
```bash
flutter pub get
```

### 4. 运行应用
```bash
# 查看可用设备
flutter devices

# 运行（会自动选择设备）
flutter run
```

### 5. 或者在 IDE 中运行
- **VS Code**: 按 `F5`
- **Android Studio**: 点击绿色运行按钮
